查询数据库

新建数据库

use 数据库名字
db.表名字.insert({“name”:”123”})

查询表，先进入数据库

use 数据库名字
show collection //查询有哪些表
db.表名字.find()

查找年龄大于、小于、大于等于、小于等于20

db.表名字.find({“age”:{$gt:20}})
db.表名字.find({“age”:{$lt:20}})
db.表名字.find({“age”:{$gte:20}})
db.表名字.find({“age”:{$lte:20}})

模糊查询

db.表名字.find({“title”:/装/})

查询name以tan开头

db.表名字.find({“name”:/^tan/})

只查询一个字段

db.表名字.find({},{age:1})

查询age大于20 的name字段

db.表名字.find({“age”:{$gt20}},{name:1})

查询按照年龄排序1是升序，-1是降序

db.表名字.find({}).solt({“age”:1})
db.表名字.find({}).solt({“age”:-1})

分页用的,一页有两条数据

db.表名字.find().skip(0)..limit(2)
db.表名字.find().skip(2)..limit(2)
db.表名字.find().skip(4)..limit(2)

查询age20或者age23

db.表名字.find({$or:[{“age”:20},{“age”:23}]})
